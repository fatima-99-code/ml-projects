# Machine Learning Projects

This repository contains 4 end-to-end Machine Learning projects built using Python, Scikit-learn, XGBoost, Pandas, NumPy, Matplotlib, and Seaborn.

These projects cover:
- Classification
- Regression
- Time Series Forecasting
- Feature Engineering
- Model Evaluation
- Hyperparameter Tuning
- Deployment Preparation

---

# Projects Included

## 1. HR Attrition Prediction
Predict employee attrition using machine learning models.

### Key Features
- Data preprocessing
- SMOTE for class imbalance
- Feature scaling
- Logistic Regression
- Random Forest
- Gradient Boosting
- ROC-AUC evaluation
- Attrition risk prediction

### Goal
Help HR teams identify employees likely to leave the company.

---

## 2. Customer Churn Prediction
Predict telecom customer churn behavior.

### Key Features
- Data cleaning
- One-hot encoding
- Feature engineering
- Model comparison
- ROC Curve analysis
- Feature importance analysis
- Prediction pipeline saving

### Goal
Help telecom companies improve customer retention.

---

## 3. Credit Risk Scoring
Predict loan default risk using classification models.

### Key Features
- Missing value handling
- Label encoding
- SMOTE balancing
- Random Forest & Gradient Boosting
- Hyperparameter tuning with GridSearchCV
- ROC-AUC evaluation

### Goal
Assist financial institutions in assessing customer credit risk.

---

## 4. Sales Forecasting
Forecast future Walmart sales using machine learning and time-series features.

### Key Features
- Time-series preprocessing
- Lag features
- Rolling statistics
- Linear Regression
- Random Forest
- XGBoost
- Time-based train/test split
- Future sales forecasting

### Goal
Help businesses predict future sales trends for planning and inventory management.
